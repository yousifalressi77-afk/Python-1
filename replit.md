# Discord Bot & Dashboard System

## المشروع (Project Overview)
نظام بوت Discord عالمي مع لوحة تحكم وأمان وحماية وكريدت نظام.
Global Discord bot system with dashboard, security, credit system, and admin commands.

## البنية (Architecture)
- **Bot Backend**: Discord.js bot with slash commands
- **Dashboard API**: Express.js REST API on port 5000
- **Database**: PostgreSQL for user data, credits, and guild settings
- **Security**: JWT authentication, bcrypt hashing, rate limiting

## المميزات (Features)
- ✅ Global bot with slash commands
- ✅ Credit/balance system
- ✅ Dashboard with admin controls
- ✅ User management
- ✅ Command configuration
- ✅ Security & protection

## التكوين (Configuration)
- Language: Node.js (JavaScript)
- Database: PostgreSQL
- Required Secret: DISCORD_TOKEN
